How it Works

Create employee(project user) and assign related user with
the access rights of project user.


After logging in as a project user, user can see the
"Fill up Timesheet" button in the
list view of tasks/issues.


With this button, user can fill up the timesheet according
to his tasks/issues easily.


Once the user submits the timesheet, user cannot edit
or delete the timesheet entry.


Users and managers need to make sure that the
"Fill up Timesheet" button visibility is maintained
from Stages configuration.
