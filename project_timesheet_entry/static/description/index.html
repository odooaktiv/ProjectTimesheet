Project Timesheet Entry

Using this module, registered project user can fill up timesheet in project tasks and issues from both list and form view on click of a button. User can also view the stage of work (To do, In Progress, Cancelled, Done). Here are the features of this module:

  • “Fill up Timesheet” button will allow to enter the duration for allotted work which will reflect in the timesheet tab.
  • Also, from configuration, users and managers can manage the visibility of “Fill up Timesheet” button in accordance with task stages.
  • Once submitted the timesheet can’t be edited or deleted.

Prerequisites before Installation

No prerequisites are required before the installation.

Configurations

There are no Configuration required after the installation.

Create employee (project user) and assign related user with the access rights of project user.

The button shown in the image-“Fill up Timesheet” will allow user to provide duration for specific task and issues in list view.

After clicking, button will navigate user to this wizard where they can mention the details of work log and allocate specific time limit for the task/issue.

Once saved, the timesheet entry can’t be edited/deleted by user. The same thing is possible for the project managers but the main difference is that even after saving the timesheet, managers can edit/delete the entries.

The users/managers can manage the visibility of Timesheet button in every stage from Stages configuration. Go to > configuration tab > Click on stages attribute and select the stage for project.
